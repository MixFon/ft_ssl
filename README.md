# ft_ssl
В данном проекте реализован алгоритм шифрования des.

Режимы des:
* des-ecb
* des-cbc
* des3
* des-pcbc
* des-ofb
* des-cfb
* des-ctr
* simple

Так же реализованы алгоритмы хеширования md5 и sha256

## Использование
    git clone https://github.com/MixFon/ft_ssl.git
    cd ft_ssl
    git clone https://github.com/MixFon/libft.git libft
    make
